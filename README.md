# snakeGame

Basic snake game with local multiplayer support upto 4 players.

## Getting started

You can clone the repo to play the game on your local machine or play online by clicking [here](codervivek.github.io/snakeGame) .
### Default controls

##### Player 1
Speed : Up Arrow
Bomb : Down Arrow
Left : Left Arrow
Right : Right Arrow
##### Player 2
Speed : w
Bomb : s
Left : a
Right : d
##### Player 3
Speed : u
Bomb : k
Left : j
Right : l
##### Player 4
Speed : Numpad 8
Bomb : Numpad 2
Left : Numpad 4
Right : Numpad 6
